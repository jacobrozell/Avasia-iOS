# Avasia — Foreshadowing & Saga Payoff Map

> Room-by-room guide for planting **§0 sacred pillars** ([`SAGA.md`](SAGA.md)) without
> info-dumping. KoN = wonder + dread; SoC = war + compromise; game 3 = price + cult.
>
> **Rule:** foreshadowing is **optional** (LOOK, ASK, off-path TALK) unless noted as
> critical-path beats. Veterans and re-readers catch the thread.
>
> **Continuity holes:** see [`NARRATIVE_LOGIC.md`](NARRATIVE_LOGIC.md) before
> adding new seeds — don't foreshadow what isn't decided yet.

## Voice rules

1. **KoN:** prophetic, not preachy — no "Commodity Era" or "anchor law" by name.
2. **SoC:** exhausted honesty — winners name the cost on the critical path.
3. Use *tied*, *bound*, *gift*, *ledger*, *echo*, *chain* — not system jargon.
4. **§0** is law — see [`SAGA.md`](SAGA.md) before adding new pillars.
