# Story Hub

## Metadata

| Field | Value |
|---|---|
| **Swift `AnthologyRoomID`** | `storyHub` |
| **Display name** | Story Adventures |
| **Room type** | logic |
| **Region** | `splitpath` |
| **Source** | `StoryHubRoom.swift` |

## Purpose

Meta hub after New Game / between stories. Shows FP, alignment, and unlock status.

## Commands

| Input | Outcome |
|---|---|
| `LIST` | All stories with cost and lock reason |
| `PLAY SCOUT` | Story #0 (free; replayable) |
| `PLAY GOOD` | Good #1 — 500 FP, loyalist |
| `PLAY BAD` | Bad #1 — 500 FP, agroman |
| `PLAY ELK` | Elk Feast — 250 FP, neutral |
