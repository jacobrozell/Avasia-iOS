## How to use this list

Each entry includes a **hook**, **POV**, **pillar** (saga seed), and **priority**.

| Priority | Meaning |
|---|---|
| **P0** | Fills a canon hole or feeds game 3 directly |
| **P1** | Rich anthology / magazine standalone |
| **P2** | Flavor, comedy, or myth — still useful |

**Rules (do not break when drafting):**

- No barbarian reveal that vindicates Vashirr — Thekia's doubt stands.
- No mother rescue in Age era — only *why* and *where rumored* (`NARRATIVE_LOGIC` #8).
- SoC PC **lives** — game 3 uses myth, propaganda, or cameo — not resurrection-villain.
- Earnest default — comedy in Athalos, priestess *42*, orange fish; massacre and memory stay clean.

## World-building buckets

### Geography not yet "lived in"

| Place | What's missing | Story / lore use |
|---|---|---|
| **Ofelos** | Almost no interior scenes | Neutral city: draft law, crystal tax, "united Avasia" bureaucracy |
| **Agromanian heartland** (west) | Enemy camp as set piece only | Brotherhood oath culture; middle name as banner |
| **Varatro Falls** | Tomb quest, thin daily life | Pilgrimage, first king myth, Blade recovery ecology |
| **Northern march** | Barbarian lie never pays off on-page | Show the horde that **never came** |
| **Aylova underbelly** | Capital as rally only | Refugee quarters, mage housing crisis, Ring toll riots |
| **Oceandale pre-burn** | Fishing colony economy | Seasonal fleet, civic church, ground-house registry |
| **NW cave prison history** | KoN flavor only | Pre–Old-Mage jail; why spells hidden in punishment site |

### Institutions to name and staff

| Institution | One-line function | Fiction use |
|---|---|---|
| **High Mage's Council** (reformed) | Public spell-glass, not secret cabal | Thekia post-KoN politics |
| **Chronicler ranks** | Saga XP diegetic layer | Historians argue over KoN/SoC events |

| **Ring Road Survey** | Post-war infrastructure | Splitpath industrialized; toll clerks, smugglers |
| **Crystal Line Company** | Anula relay + price board | Ofelos line updates market; surveillance |
| **Druid Legion (reborn)** | SoC epilogue | Diaspora culture, Messenger myth, relic trade |
| **Ground-house registry** | Council charter for sky-heirs | Kaefden family, Thekia backstory |

---

### Cultures & daily life (texture)

- **Mages:** pointed ears as visible minority; tower vs shore accent after scatter.
- **Druids:** shapeshift as labor (scout fox, hunter wolf); Cataracta class as caste.
- **Sylvians:** hunt-god church vs elder secular law; north migration grief.
- **Agromanians:** brotherhood oath meals; Many Hands as *found religion* for soldiers.
- **Neutrals:** Ofelos coin accepted everywhere; resentment when neutrality fails.
- **Fishers:** Oceandale shore practice — meditation/yoga as real custom (optional comedy).
