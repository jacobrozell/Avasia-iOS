B-02 — The Mother's Thread

Field Value
Canon anchor NARRATIVE_LOGIC #8 · game 3 mother-search
POV Epistolary — Thekia's filed letters, registry notes, never sent
Length target 3,500–4,500 words
When Years 1–7 post-KoN
Cast Thekia; clerks; Agromanian registry fragments; Kaefden (addressee, never reads)

Synopsis

Thekia builds a mother-search archive without promising rescue. Letters
gather: ground-oath clause copy; Agromanian camp lists with one name circled;
flesh-forge rumor from coalition scouts; a witness who saw portal east before
Oceandale burned. Each entry ends unresolved. Age-era rule: no reunion,
no body, no heroic extraction.

Scene beats

  1. File opened — pendant registry cross-reference.
  2. Camp list — name partial; ink smeared.
  3. Scout rumor — flesh-anchor stock; Thekia vomits, keeps filing.
  4. Letter to Kaefden (unsent) — "I will not give you hope without proof."
  5. Year seven — archive sealed for game 3; one red string on the wall.
  6. Coda — spell-glass note: "Search continues. Rescue deferred."

Theme

Mystery as responsibility — clues without catharsis.

Tone

Cold document tragedy.

Connections
